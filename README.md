# Exploratory Data Analysis using Tableau

## Dataset
Customer Churn Dataset

## Objective
To perform exploratory data analysis (EDA) to identify trends, patterns,
anomalies, and data quality issues using Tableau.

## Tasks Covered
- Asking meaningful business questions
- Understanding data structure and variables
- Identifying trends and patterns
- Testing hypotheses using visualizations
- Detecting data quality issues

## Tools Used
- Tableau Desktop
- GitHub
